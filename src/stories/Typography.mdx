# Typography

Font sizes used in the project. All sizes are fluid: they grow smoothly between mobile and desktop using `clamp()`.

Font family: **Space Grotesk** (300 to 700).

## How fluid typography works

`clamp(min, preferred, max)` returns the preferred value (based on viewport width) but never goes below min or above max. This way text size grows naturally with screen width without big jumps at breakpoints.

Try resizing the browser window to see how sizes change.
