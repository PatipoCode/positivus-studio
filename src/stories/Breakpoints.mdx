## Usage

```scss
.element {
  font-size: 24px;

  @media (max-width: $bp-md) {
    font-size: 18px;
  }
}
```

## When to use breakpoints

The project mostly uses fluid sizing with `clamp()` because it scales smoothly. Breakpoints are needed only when layout needs to change. For example: switching flex direction from row to column, changing grid columns from 3 to 1.
